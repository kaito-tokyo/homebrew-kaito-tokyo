# Kaito-tokyo Kaito-tokyo

## How do I install these formulae?

`brew install kaito-tokyo/kaito-tokyo/<formula>`

Or `brew tap kaito-tokyo/kaito-tokyo` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kaito-tokyo/kaito-tokyo"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
